# Portafolio Tailwind (estático)

Proyecto de ejemplo de un portafolio de desarrollador usando **Tailwind CSS (CDN)**.
Incluye secciones: Inicio, Proyectos, Habilidades, Experiencia y Contacto.

## Inicio rápido

1. Descarga y descomprime el ZIP.
2. Abre `index.html` en tu navegador.
3. Edita los textos, enlaces e ítems de proyectos a tu gusto.
4. (Opcional) Cambia `resume.html` con tu información real.

> Nota: el uso del CDN de Tailwind es solo para demos. Para producción, configura Tailwind con PostCSS y genera CSS optimizado.
